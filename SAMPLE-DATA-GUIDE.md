# 📊 엑셀 데이터 샘플 파일 가이드

보험사 시책 계산기에 데이터를 입력하기 위한 샘플 파일입니다.

## 📁 제공된 샘플 파일

1. **sample-data-db-week1.txt** - DB손보 1주차 샘플 데이터
2. **sample-data-heungkuk-week1.txt** - 흥국화재 1주차 샘플 데이터
3. **sample-excel-template.xlsx** - 엑셀 템플릿 (아래 방법으로 생성)

## 🎯 사용 방법

### 1단계: 샘플 파일 열기

제공된 `.txt` 파일을 텍스트 에디터나 엑셀에서 열기:
- Windows: 메모장 또는 Excel
- Mac: TextEdit 또는 Excel

### 2단계: 데이터 복사

1. 파일 전체 선택 (Ctrl+A / Cmd+A)
2. 복사 (Ctrl+C / Cmd+C)

### 3단계: 계산기에 붙여넣기

1. 배포된 웹사이트 접속
2. "⚙️ 관리자 페이지" 클릭
3. 비밀번호 입력: `admin1234`
4. **"인별 데이터 관리"** 탭 선택
5. 보험사 선택 (예: DB손보)
6. 주차 선택 (예: 1주차)
7. 텍스트 영역에 **붙여넣기** (Ctrl+V / Cmd+V)
8. "💾 데이터 파싱 및 저장" 클릭
9. 미리보기에서 데이터 확인
10. "💾 전체 저장" 클릭

### 4단계: 다른 주차/보험사 반복

- 2주차, 3주차, 4주차 데이터도 같은 방식으로 입력
- 다른 보험사(흥국화재, 메리츠, KB손보, 이달주력사)도 동일

## ⚠️ 중요: 데이터 형식

### 필수 컬럼 (반드시 포함!)

```
순위	이름	사업부	본부	지점	[채널]	영업시작월	영업차월	계약건수	월납보험료	평가업적
```

### 컬럼 설명

| 컬럼명 | 필수 | 설명 | 예시 |
|--------|------|------|------|
| 순위 | ✓ | 순위 번호 | 1, 2, 3... |
| 이름 | ✓ | 사원 이름 | 김진훈 |
| 사업부 | ✓ | 소속 사업부 | 디지털3사업부 |
| 본부 | ✓ | 소속 본부 | 디지털강북본부 |
| 지점 | ✓ | 소속 지점 | 강북4지점 |
| [채널] | - | DB채널, 흥국채널 등 | 보장분석 |
| 영업시작월 | - | 영업 시작 월 | 2021/08 |
| 영업차월 | - | 영업 개월 수 | 54.0 |
| 계약건수 | - | 계약 건수 | 26 |
| **월납보험료** | ✓ | **시책금 계산 기준!** | 2222608 |
| 평가업적 | - | 평가 업적 | 2061620 |

### 🔴 핵심: "월납보험료" 컬럼

**시책금 계산은 "월납보험료" 컬럼 값을 사용합니다!**

- 원 단위로 입력: `2222608` (2,222,608원)
- 계산기는 자동으로 만원 단위로 변환: `222만원` (소수점 버림)
- 예: 
  - `2222608` → `222만원`
  - `1199999` → `119만원`
  - `5000000` → `500만원`

## 📝 실제 엑셀에서 데이터 준비하는 방법

### 방법 1: 엑셀에서 직접 작성

```
1. 새 엑셀 파일 열기
2. 첫 번째 행에 컬럼명 입력:
   순위 | 이름 | 사업부 | 본부 | 지점 | DB채널 | ... | 월납보험료 | 평가업적
3. 두 번째 행부터 데이터 입력
4. 전체 선택 → 복사 (Ctrl+C)
5. 계산기에 붙여넣기
```

### 방법 2: 기존 엑셀 파일 사용

```
1. 기존 업적 엑셀 파일 열기
2. 필수 컬럼만 선택:
   - 이름
   - 본부
   - 지점
   - 월납보험료 (중요!)
3. 컬럼 순서 맞추기
4. 복사 → 계산기에 붙여넣기
```

## 💡 실전 예시

### DB손보 1주차 (1/1~1/8) 데이터 입력

```txt
순위	이름	사업부	본부	지점	DB채널	영업시작월	영업차월	계약건수	월납보험료	평가업적
1	김진훈	디지털3사업부	디지털강북본부	강북4지점	보장분석	2021/08	54.0	26	2222608	2061620
2	이수민	디지털1사업부	디지털강남본부	강남1지점	보장분석	2020/03	60.0	18	1850000	1820000
```

**결과:**
- 김진훈: 222만원 → 시책금 자동 계산
- 이수민: 185만원 → 시책금 자동 계산

### 흥국화재 2주차 (1/9~1/15) 데이터 입력

```txt
순위	이름	사업부	본부	지점	흥국채널	영업시작월	영업차월	계약건수	월납보험료	평가업적
1	박민준	디지털2사업부	디지털서초본부	서초2지점	온라인	2019/05	68.0	22	3150000	3100000
```

**결과:**
- 박민준: 315만원 → 시책금 자동 계산

## 🔄 전주차 (전월) 데이터란?

**전주차**는 1주차의 **연속가동 시상 계산**을 위한 참고 데이터입니다.

### 사용 예시

```
전주차(전월 마지막 주차): 김진훈 30만원
1주차: 김진훈 20만원

→ 연속가동 조건 충족 시 추가 시상금 지급!
```

### 입력 방법

1. 관리자 페이지 → 인별 데이터 관리
2. 주차 선택: **"전주차(전월)"** 선택
3. 전월 마지막 주차 데이터 붙여넣기
4. 저장

## 📊 전체 워크플로우

```
1. DB손보 전주차 데이터 입력
   ↓
2. DB손보 1주차 데이터 입력
   ↓
3. DB손보 2주차 데이터 입력
   ↓
4. DB손보 3주차 데이터 입력
   ↓
5. DB손보 4주차 데이터 입력
   ↓
6. 💾 전체 저장
   ↓
7. 흥국화재 전주차~4주차 반복
   ↓
8. 메리츠화재 전주차~4주차 반복
   ↓
9. KB손보 전주차~4주차 반복
   ↓
10. 이달주력사 전주차~4주차 반복
   ↓
11. 💾 최종 전체 저장
   ↓
12. 📥 백업 다운로드 (중요!)
```

## 🎯 체크리스트

입력 전 확인사항:

- [ ] 컬럼명이 포함되어 있는가?
- [ ] "월납보험료" 컬럼이 있는가?
- [ ] 데이터가 탭으로 구분되어 있는가?
- [ ] 사원 이름이 정확한가?
- [ ] 본부/지점 정보가 정확한가?

입력 후 확인사항:

- [ ] 미리보기에서 데이터가 정확한가?
- [ ] 총 인원수가 맞는가?
- [ ] "전체 저장"을 클릭했는가?
- [ ] 백업 파일을 다운로드했는가?

## ⚠️ 주의사항

### 1. 탭 구분자 필수

엑셀에서 복사한 데이터는 자동으로 탭으로 구분됩니다.
텍스트 파일에서 직접 작성할 때는 반드시 **Tab 키**로 구분하세요!

❌ 잘못된 예 (공백):
```
순위 이름 사업부 본부 지점
```

✅ 올바른 예 (탭):
```
순위	이름	사업부	본부	지점
```

### 2. 월납보험료 단위

- 원 단위로 입력: `2222608`
- 콤마 없이 숫자만: ~~`2,222,608`~~ → `2222608`

### 3. 데이터 중복 방지

같은 보험사의 같은 주차 데이터를 여러 번 입력하면 **덮어씌워집니다**.
- 기존 데이터가 사라지므로 주의!

### 4. 백업 필수

데이터 입력 후 반드시 백업:
1. 관리자 페이지 → "📥 인별 데이터 다운로드"
2. JSON 파일 안전하게 보관

## 🆘 문제 해결

### "데이터 파싱 실패" 오류

**원인**: 잘못된 형식
**해결**:
1. 컬럼명이 제대로 있는지 확인
2. 탭 구분자인지 확인 (공백 X)
3. 샘플 파일과 동일한 형식인지 비교

### "월납보험료가 0으로 나옴"

**원인**: 월납보험료 컬럼이 없거나 위치가 틀림
**해결**:
1. 컬럼명이 정확히 "월납보험료"인지 확인
2. 컬럼 순서가 샘플과 같은지 확인

### "사원이 검색되지 않음"

**원인**: 전체 저장을 안 했거나 localStorage 문제
**해결**:
1. "💾 전체 저장" 버튼 클릭
2. 페이지 새로고침 (F5)
3. 여전히 안 되면 localStorage 확인
