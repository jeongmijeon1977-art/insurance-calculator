⚠️ 중요 안내

1. "월납보험료" 컬럼이 중요합니다!

시책금 계산은 "월납보험료" 컬럼 값을 기준으로 합니다.

2. 원 단위로 입력하세요

예: 2,222,608원 → 2222608 (콤마 제거)

3. 탭으로 구분되어야 합니다

엑셀에서 복사하면 자동으로 탭 구분됩니다.

4. 백업을 꼭 하세요!

데이터 입력 후 관리자 페이지에서 "📥 인별 데이터 다운로드" 클릭

📝 실제 엑셀 데이터 만드는 방법

방법 1: 엑셀에서 직접 작성

  1. 새 엑셀 파일 열기
  2. 첫 번째 행에 컬럼명 입력 (위 샘플 참고)
  3. 두 번째 행부터 사원 데이터 입력
  4. 전체 선택 (Ctrl+A) → 복사 (Ctrl+C)
  5. 계산기 관리자 페이지에서 붙여넣기 (Ctrl+V)

방법 2: 기존 엑셀 파일 사용

  1. 기존 업적 엑셀 파일 열기
  2. 필수 컬럼 확인: 이름, 본부, 지점, 월납보험료
  3. 컬럼 순서를 위 샘플과 동일하게 정렬
  4. 데이터 범위 선택 → 복사
  5. 계산기에 붙여넣기

방법 3: 샘플 데이터 수정

  1. 위 샘플 데이터 복사
  2. 엑셀에 붙여넣기
  3. 이름과 금액만 실제 데이터로 수정
  4. 다시 복사 → 계산기에 붙여넣기
